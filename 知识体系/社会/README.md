## A股短线交易策略

### 选股模型（2026-04-23 第一版）

#### 筛选条件
1. **量比异动**：量比 > 3
2. **涨幅区间**：今日涨幅 3%-7%
3. **换手率**：5%-25%
4. **资金流入**：净流入 > 1亿
5. **板块效应**：所属板块当日涨幅前10

#### 排除条件
- ST/*ST股票
- 换手率>50%
- 连续涨停后高位放量
- **排除创业板**（代码以300开头）

#### 交易规则
- **买入时间**：周一至周四 14:30-15:00
- **禁止买入**：周五、节假日前一天
- **卖出时间**：次日 9:30-10:30
- **周五规则**：周五早盘必须卖出，下午不买入
- **止盈**：+3% / **止损**：-2%
