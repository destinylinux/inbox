# 西方经济学核心内容

> 本文梳理西方经济学从重商主义到行为经济学的主要流派，涵盖16世纪至今约500年的理论演进。

---

## 一、重商主义（16-18世纪）

| 项目 | 内容 |
|------|------|
| **历史背景** | 地理大发现、资本原始积累、民族国家兴起 |
| **核心观点** | 财富即金银（贵金属）；财富来源在流通领域（对外贸易）；国内贸易不增加国家总财富 |
| **政策主张** | **奖出限入**：追求贸易顺差，高关税保护本国产业，政府全面干预经济|
| **代表人物** | 托马斯·曼（Thomas Mun，1571-1641，英国）、柯尔贝尔（Jean-Baptiste Colbert，1619-1683，法国）、孟克列钦（法国） |
| **代表著作** | 托马斯·曼《英国得自对外贸易的财富》|
| **主要贡献** | 推动了早期国际贸易理论的形成；促进了民族国家的工商业发展和殖民扩张 |
| **主要争议** | 将金银等同于财富本末倒置（后世称之为"**重商主义谬误**"）；零和博弈思维忽视了互利共赢的贸易本质；过度干预扭曲了市场机制 |
| **阶段划分** | 早期（重金主义）—禁止货币出口；晚期（重工主义）—追求贸易顺差、发展本国制造业 |

---

## 二、古典经济学（18-19世纪）

古典经济学是经济学的奠基时期，核心主张是自由放任、市场自发调节。

### 2.1 亚当·斯密（Adam Smith，1723-1790）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《国民财富的性质和原因的研究》（简称《国富论》，1776年）|
| **核心理论** | ① **分工理论**：分工提高劳动生产率，是国民财富增长的根本原因（扣针工厂的著名例子）；② **"看不见的手"**：个人追求自身利益时，在市场机制引导下无意中促进了社会整体利益；③ **绝对优势理论**：各国应生产并出口绝对成本低于他国的商品；④ **劳动价值论雏形**：劳动是衡量一切商品交换价值的真实尺度 |
| **主要贡献** | 奠定了现代经济学的学科基础；批判重商主义，系统论证了自由贸易和自由市场的优越性 |
| **经典名言** | "我们期望的晚餐并非来自屠夫、酿酒师和面包师的恩惠，而是来自他们对自身利益的关切。" |

### 2.2 大卫·李嘉图（David Ricardo，1772-1823）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《政治经济学及赋税原理》（1817年）|
| **核心理论** | ① **比较优势理论**：即使一国在所有商品生产上都处于绝对劣势，仍可通过专业化生产相对成本更低的商品并参与国际贸易而获利（"两优取其重，两劣取其轻"）；② **分配理论**：研究地租、工资、利润三者之间的分配规律；③ **劳动价值论深化**：进一步发展了斯密的劳动价值思想 |
| **主要贡献** | 比较优势理论成为现代国际贸易理论的基石；证明了自由贸易对参与各方均有利，有力批驳了保护主义 |
| **争议** | 劳动价值论遇到"不变的价值尺度"难题（后来马克思试图解决）；理论假设与现实条件之间存在差距 |

### 2.3 托马斯·马尔萨斯（Thomas Malthus，1766-1834）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《人口原理》（1798年）|
| **核心观点** | 人口以**几何级数**增长，生活资料以**算术级数**增长；人口增长必然超出生活资料供给，导致贫困、饥荒和战争——即"**马尔萨斯陷阱**" |
| **主要贡献** | 提出人口与资源约束之间的基本问题，启发了后来的发展经济学和资源经济学 |
| **主要争议** | 预言被技术革命（农业生产力大幅提升）和人口转型（出生率随发展下降）打破；被批评为为富人不愿救济穷人提供理论辩护 |

### 2.4 约翰·斯图尔特·穆勒（John Stuart Mill，1806-1873）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《政治经济学原理》（1848年）— 19世纪中叶经济学标准教科书 |
| **核心贡献** | ① 综合并集成了古典经济学的全部成果；② 区分了"**生产规律**"（自然规律、难以改变）和"**分配规律**"（可人为调整、包含社会选择）；③ 最早关注社会公正问题，主张累进税、缩短工时、促进女性权利——是古典经济学向新古典过渡的桥梁 |
| **独特之处** | 既是经济学家也是哲学家和自由主义者，思想兼具自由与进步双重属性 |

## 三、马克思主义经济学（19世纪）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《资本论》（第一卷1867年） |
| **核心理论** | 劳动价值论 → 剩余价值论 → 资本积累理论 → 资本主义灭亡的历史必然性（详见《中国经济学/README.md》第二部分）|
| **理论定位** | 既是古典经济学的批判继承，也是超越古典的新范式——从"解释资本主义"转向"批判资本主义并预见社会主义" |
| **核心批判** | 古典经济学将资本主义视为永恒的自然秩序；马克思揭示其历史暂时性和内在矛盾 |
| **对后世影响** | 直接催生了20世纪社会主义国家的经济实践；启发了发展经济学、依附理论、世界体系理论；至今是批判资本主义的最有力理论武器 |

## 四、新古典经济学（19世纪末-20世纪）

### 4.1 边际革命

| 项目 | 内容 |
|------|------|
| **发生时间** | 19世纪70年代，几乎同时在三国独立发生 |
| **代表人物** | 卡尔·门格尔（Carl Menger，奥地利）、威廉·斯坦利·杰文斯（W.S. Jevons，英国）、莱昂·瓦尔拉斯（Léon Walras，法国） |
| **核心创新** | **边际效用价值论**：商品价值不是由生产它所耗费的劳动决定，而是由消费者对最后一单位商品的主观评价（**边际效用**）决定 |
| **理论意义** | 以**主观价值论**取代古典经济学和马克思主义的**劳动价值论**；引入了**边际分析**（微积分）这一数学工具，使经济学成为更"科学"的学科 |
| **重要分支** | 门格尔创立了**奥地利学派**；瓦尔拉斯建立了**一般均衡理论** |

### 4.2 阿尔弗雷德·马歇尔（Alfred Marshall，1842-1924）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《经济学原理》（1890年）— 主导西方经济学教材半个世纪 |
| **核心贡献** | ① **供求均衡价格论**：价格由供给和需求共同决定，如同剪刀的两片刀刃（"马歇尔剪刀"）；② **消费者剩余**与**生产者剩余**概念；③ 将边际分析系统化、全面引入微观经济学；④ 区分**短期**与**长期**分析 |
| **理论综合** | 综合了古典生产成本论和边际效用价值论——供给方由生产成本决定、需求方由边际效用决定，价格在均衡点形成 |
| **学科称呼** | 将"政治经济学"改称为"**经济学**"（Economics），弱化了政治维度，强调科学性 |

## 五、凯恩斯主义（20世纪30年代）

### 背景

1929-1933年大萧条彻底击碎了新古典经济学"市场自动实现充分就业"的信念，为凯恩斯的革命性理论创造了历史土壤。

### 约翰·梅纳德·凯恩斯（John Maynard Keynes，1883-1946）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《就业、利息和货币通论》（1936年）— 简称《通论》 |
| **核心理论** | **有效需求不足论**：资本主义经济的常态不是充分就业均衡，而是**有效需求不足** |
| **三大心理规律** | ① **边际消费倾向递减**：收入增加，消费增量递减（储蓄增加）；② **资本边际效率递减**：投资增加，预期回报率下降；③ **流动偏好**：人们偏好持有现金，利率不能无限下降 |
| **政策主张** | ① **政府干预**：通过扩张性财政政策（增加公共支出、减税）弥补私人投资不足；② **举债操作**：赤字财政在萧条期是必要的；③ **货币政策**：降低利率刺激投资 |
| **乘数效应** | 政府支出增加会通过连锁消费带动数倍于初始支出的总需求增长 |
| **核心观点** | "**长期中我们都死了**" — 凯恩斯关注短期的需求管理，不认同"市场长期会自动调整"的古典立场 |

### 主要贡献与争议

| 贡献 | 争议 |
|------|------|
| 开辟了**宏观经济学**这一独立学科分支 | 无法解释1970年代的"**滞胀**"（高通胀+高失业并存） |
| 为政府干预经济提供了系统理论依据 | 三大心理规律建立在主观心理基础上，缺乏微观基础 |
| 深刻影响了战后西方国家的经济政策（充分就业目标） | 短期政策长期化导致政府债务膨胀 |

## 六、新古典综合派（20世纪中叶-今）

| 项目 | 内容 |
|------|------|
| **代表人物** | 保罗·萨缪尔森（Paul Samuelson，1915-2009）、詹姆斯·托宾、罗伯特·索洛、弗兰科·莫迪利安尼 |
| **代表著作** | 萨缪尔森《经济学》（1948年首版）— 史上最畅销的经济学教科书 |
| **核心主张** | 将凯恩斯的**宏观经济分析**与马歇尔的新古典**微观经济学**综合为统一的经济学体系 |
| **理论框架** | IS-LM模型（希克斯-汉森）→ AD-AS模型（总需求-总供给）→ 菲利普斯曲线（通胀与失业的权衡） |
| **"混合经济"论** | 当代发达国家既非纯粹自由市场也非计划经济，而是市场机制与国家干预相结合的**混合经济** |
| **政策主张** | **相机抉择**的财政政策和货币政策：经济萧条时扩张、过热时紧缩 |
| **衰落原因** | 1970年代"滞胀"无法用凯恩斯理论解释——菲利普斯曲线失效 |
| **历史意义** | 在二战后30年间占据西方经济学正统地位；塑造了现代宏观经济学的基本分析工具 |

## 七、货币主义（20世纪50-80年代）

| 项目 | 内容 |
|------|------|
| **代表人物** | 米尔顿·弗里德曼（Milton Friedman，1912-2006）|
| **代表著作** | 《资本主义与自由》（1962年）、《美国货币史》（与施瓦茨合著，1963年）|
| **核心主张** | ① **货币最重要**：货币供给量是决定名义收入和物价水平的主导因素；② **通货膨胀归根结底是货币现象**；③ 反对凯恩斯的相机抉择政策，主张**单一货币规则**（货币供给按固定比例增长）|
| **理论依据** | ① **永久收入假说**：消费不取决于当期收入而取决于长期平均预期收入；② **自然失业率**：长期内失业率由实际因素决定，货币政策无法永久降低失业率 |
| **政策影响** | 深刻影响了20世纪80年代美国（里根）和英国（撒切尔）的经济政策转向——控制货币供给、减少政府干预 |
| **与凯恩斯分歧** | 凯恩斯强调**财政政策**（政府支出），货币主义强调**货币政策**（货币供给）；凯恩斯主张积极干预，货币主义主张规则化、减少干预 |
| **主要贡献** | 重新确立了货币在经济分析中的核心地位；挑战了凯恩斯主义的政策主导地位 |

## 八、新制度经济学（20世纪60年代-今）

| 项目 | 内容 |
|------|------|
| **代表人物** | 罗纳德·科斯（Ronald Coase，1910-2013）、道格拉斯·诺斯（Douglass North，1920-2015）、奥利弗·威廉姆森（Oliver Williamson，1932-2020）、张五常 |
| **核心创新** | 在新古典经济学框架内引入**制度分析**，修正"零交易费用"的理想化假设|

### 8.1 科斯 — 交易成本与产权

| 理论 | 内容 |
|------|------|
| **交易成本** | "利用价格机制的费用"——包括搜寻信息、讨价还价、订立和执行合同的全部成本。企业之所以存在，正是因为内部管理成本低于市场交易成本 |
| **科斯定理** | 在交易成本为零且产权清晰的前提下，无论初始产权如何分配，市场交易都能达到最优资源配置。**反之**：如果交易成本为正，法律制度对资源配置就至关重要 |
| **企业理论** | 企业是对市场的替代——当市场交易成本高于企业内部协调成本时，企业就出现了；两种模式的边际成本相等时为企业规模边界 |

### 8.2 诺斯 — 制度变迁

| 理论 | 内容 |
|------|------|
| **制度定义** | 社会的博弈规则——正式规则（法律、宪法）和非正式约束（习俗、道德）的总和 |
| **国家理论** | 国家是"在行使暴力上有比较利益的组织"，处于规定和实施产权的地位——这便是著名的"**诺斯悖论**"：没有国家办不成事，有了国家又可能产生低效率的产权 |
| **路径依赖** | 制度变迁受到历史先例和沉没成本的约束，低效率的制度可能长期存在（如QWERTY键盘） |

### 8.3 核心意义

新制度经济学架起了制度、交易成本与新古典理论之间的桥梁，使经济学能够分析法律、企业组织、社会文化等真实世界中的制度现象。它对理解为什么不同国家有不同经济绩效提供了关键分析框架。

## 九、行为经济学（20世纪70年代-今）

| 项目 | 内容 |
|------|------|
| **学科定位** | 将心理学研究成果引入经济分析，修正新古典"完全理性人"假设 |
| **核心武器** | 实验方法、认知心理学 |

### 9.1 卡尼曼与前景理论（Kahneman & Tversky）

| 项目 | 内容 |
|------|------|
| **代表人物** | 丹尼尔·卡尼曼（Daniel Kahneman，1934-2024）、阿莫斯·特沃斯基（Amos Tversky，1937-1996）|
| **核心发现** | **前景理论**：人们在做决策时，对**损失**的敏感程度远高于对**同等收益**的敏感程度（损失厌恶）——失去100元的痛苦约是得到100元快乐的两倍 |
| **重要效应** | ① **框架效应**：问题的表述方式影响决策；② **禀赋效应**：一旦拥有某物，对其估值会立刻提高；③ **确定性效应**：人们高估确定性的小收益，低估可能性的损失 |

### 9.2 理查德·塞勒与助推（Richard Thaler）

| 项目 | 内容 |
|------|------|
| **代表著作** | 《"错误"的行为》《助推》（Nudge）|
| **核心贡献** | ① **有限理性、有限意志力、有限自利**的系统性发现；② **助推（Nudge）**：不强制、不禁止，通过巧妙设计选择环境引导人们做出更好决策（如自动加入养老金计划而非手动注册）；③ **心理账户**：人们将不同来源或用途的钱放在不同的"心理账户"中，不按经济理性处理 |

### 9.3 学科意义

| 贡献 | 影响 |
|------|------|
| 推翻"完全理性人"假设 | 证明人类决策系统性地偏离理性预期 |
| 解释了大量市场异象 | 如股市过度波动、处置效应（过早卖出赢家、长期持有输家）|
| 影响公共政策 | 英国、美国等设立"助推小组"设计更有效的政策干预 |
| 温和的革命 | 不是推翻主流经济学，而是补充和修正 |

## 十、奥地利学派（19世纪末-今）

| 项目 | 内容 |
|------|------|
| **起源** | 19世纪70年代边际革命中，门格尔创立 |
| **代表人物** | 卡尔·门格尔（奠基）、路德维希·冯·米塞斯（Ludwig von Mises，1881-1973）、弗里德里希·哈耶克（Friedrich Hayek，1899-1992）|
| **核心方法论** | **主观主义**（经济学必须从行为人的感知和主观知识出发）；反对数学化和计量化 |

### 10.1 核心理论

| 理论 | 说明 |
|------|------|
| **主观价值论** | 价值完全是主观的，取决于个体对所涉商品边际效用的评价，与生产成本无关 |
| **自发秩序** | 哈耶克：市场秩序不是人为设计的产物，而是无数个体自发互动形成的——如语言、法律、货币的起源 |
| **知识问题** | 哈耶克《知识在社会中的运用》：分散在社会中的**默会知识**（地方化的、情境化的、不可量化的）无法被中央计划机关完整收集和利用——**计划经济不可行**的根本原因|
| **商业周期理论** | 政府人为压低利率导致虚假繁荣（过度投资），最终必然清算和衰退 |
| **企业家精神** | 市场是一个"发现过程"——企业家在不确定性中通过试错发现新的机会 |

### 10.2 与主流经济学的根本分歧

| 问题 | 主流经济学（新古典/凯恩斯） | 奥地利学派 |
|------|---------------------------|-----------|
| **方法论** | 均衡分析、数学模型 | 过程分析、主观理解 |
| **知识** | 假设信息充分或可量化 | 强调默会知识和知识分散性 |
| **市场** | 趋向均衡 | 不断发现和调整的开放过程 |
| **政府干预** | 必要时干预（凯恩斯、萨缪尔森）| 坚决反对（米塞斯、哈耶克）|
| **可计算性** | 可用计量经济模型预测 | 经济现象本质上是不可精确预测的 |

### 10.3 影响与争议

| 贡献 | 争议 |
|------|------|
| 深刻揭示了计划经济的信息与知识难题 | 对政府干预的极端否定被认为过于理想化 |
| 哈耶克的商业周期理论在2008年金融危机后受到重新关注 | 拒绝数学工具导致在主流经济学中被边缘化 |
| 其反集权、重自由的思想影响了20世纪自由主义运动 | 对现实经济的解释力常被质疑（如无法有效解释大萧条） |

## 第三部分：核心概念对比

### 1. 市场 vs 计划

| 维度 | 市场经济 | 计划经济 |
|------|---------|---------|
| **资源配置方式** | 价格机制、供求关系自发调节 | 中央计划机关统一指令分配 |
| **信息机制** | 分散决策，信息由价格信号传递 | 集中决策，依赖统计数据上报 |
| **激励方式** | 利润驱动、个人利益最大化 | 行政指令、政治动员 |
| **优势** | 灵活高效、创新活力强 | 可集中力量办大事（如大项目、战时经济）|
| **劣势** | 周期波动、贫富分化、公共品供给不足 | 信息不对称导致效率低下、创新动力不足 |
| **代表人物** | 斯密、哈耶克、弗里德曼 | 马克思（过渡阶段）、列宁、斯大林 |
| **当代实践** | 主要发达国家（不同程度混合） | 改革开放前中国、前苏联、古巴、朝鲜 |
| **中国特色** | **社会主义市场经济**：市场决定性作用 + 更好发挥政府作用 |

### 2. 效率 vs 公平

| 维度 | 效率优先 | 公平优先 |
|------|---------|---------|
| **核心主张** | 经济增长最大化，差距是增长的代价 | 缩小差距至上，限制贫富分化 |
| **理论基础** | 库兹涅茨倒U曲线（先分化后收敛） | 罗尔斯"差异原则"（有利于最不利者）|
| **政策倾向** | 减税、放松管制、保护产权 | 累进税、社会保障、转移支付 |
| **主要流派** | 新古典、货币主义、奥地利学派 | 马克思主义、后凯恩斯主义、福利经济学 |
| **中国立场** | 兼顾效率与公平——初次分配重效率，再分配重公平；**共同富裕**是根本目标 |

### 3. 自由放任（Laissez-faire） vs 政府干预

| 维度 | 自由放任 | 政府干预 |
|------|---------|---------|
| **核心信念** | 市场能自动趋于最优 | 市场存在系统性失灵 |
| **主要理由** | "看不见的手"、价格信号、企业家创新 | 外部性、公共品、信息不对称、垄断、周期性危机 |
| **干预范围** | 仅保护产权、维护法治（"守夜人"） | 宏观调控、产业政策、反垄断、社会保障 |
| **代表人物** | 斯密、巴斯夏、哈耶克、弗里德曼 | 凯恩斯、萨缪尔森、斯蒂格利茨 |
| **争议焦点** | 政府干预是否往往让事情更糟（"政府失灵"）| 市场失灵是否严重到需要政府介入 |
| **政策谱系** | 从古典自由主义到新自由主义 | 从凯恩斯主义到现代宏观调控 |

### 4. 劳动价值论 vs 边际效用价值论

| 维度 | 劳动价值论 | 边际效用价值论 |
|------|-----------|---------------|
| **价值来源** | **客观**——商品价值由生产它所耗费的社会必要劳动时间决定 | **主观**——商品价值取决于消费者对最后一单位的主观评价 |
| **核心主张** | 劳动是价值的唯一源泉；资本只是转移自身价值 | 价值是主观的、因人而异的，由边际效用决定 |
| **代表流派** | 古典经济学（斯密、李嘉图）→ 马克思主义政治经济学 | 边际革命（门格尔、杰文斯、瓦尔拉斯）→ 新古典经济学 |
| **解释优势** | 解释了利润的来源（剩余价值）、阶级关系和历史演进 | 解释了消费者行为、价格形成机制、资源配置 |
| **解释弱点** | 难以解释"水和钻石悖论"、无形资产定价、金融资产价格 | 无法解释利润的宏观来源、忽略了阶级关系 |
| **现实意义** | 为批判资本主义、分析剥削提供了理论武器 | 为现代微观经济学的价格理论和消费者理论奠定基础 |
| **综合视角** | 两种价值论并非全然对立——马歇尔用均衡价格论实现了形式上的综合（供给 = 生产成本，需求 = 边际效用） |

## 各流派谱系图（简略版）

```
重商主义（16-18c）
    ↓ 批判
古典经济学（18-19c）──→ 马克思经济学（19c）
    ↓ 批判与继承                     ↓
边际革命（1870s）                    社会主义政治经济学
    ↓
新古典经济学（马歇尔，1890）
    ↓ ← 大萧条冲击
凯恩斯主义（1936）──→ 新古典综合派（1940s-今）
    │                          ↑
    │                    [综合微观+宏观]
    ├── 货币主义（Friedman，1960s）
    ├── 新制度经济学（Coase，North，1960s-今）
    ├── 奥地利学派（Mises，Hayek，持续至今）
    └── 行为经济学（Kahneman，Thaler，1970s-今）
```
