## 总论

哲学是人类对世界、人生、知识、价值等根本问题的系统思考。古今中外的哲学传统虽有不同路径，但共同构成了人类智慧的宝库。

### 中国哲学脉络

- **先秦诸子**（公元前770-前221）：儒、道、墨、法、名、阴阳等
- **汉唐经学**（前206-907）：董仲舒、玄学（王弼、郭象）、佛教中国化
- **宋明理学**（960-1644）：程朱理学、陆王心学
- **清至近现代**（1644-至今）：经世致用、西学东渐、马克思主义中国化

### 西方哲学脉络

- **古希腊哲学**（约前600-公元400）：前苏格拉底、苏格拉底-柏拉图-亚里士多德、希腊化哲学
- **中世纪哲学**（400-1400）：教父哲学、经院哲学
- **近代哲学**（1400-1900）：理性主义、经验主义、德国古典哲学
- **现当代哲学**（1900-至今）：分析哲学、现象学、存在主义、后现代主义
