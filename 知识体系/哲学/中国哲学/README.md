# 知识体系 · 哲学篇 · 中国哲学

> 中国哲学核心思想家与思想
> 创建时间：2026-05-08

## 先秦诸子（公元前770-前221）

### 孔子（公元前551-前479）——儒家学派创始人

**核心思想框架：**

**① 仁**——孔子思想的核心。"仁者爱人"，"己所不欲，勿施于人"，"己欲立而立人，己欲达而达人"。仁是人的内在德性，是人与人之间应有的情感与态度。仁不是外在规范，而是发自内心的道德自觉。

**② 礼**——"克己复礼为仁"。礼是外在的行为规范和社会秩序，仁是内在的道德情感。礼以仁为根基，仁以礼为表现。孔子强调"正名"——"君君、臣臣、父父、子子"，名分不正则言不顺，言不顺则事不成。

**③ 忠恕之道**——孔子"一以贯之"之道。"忠"是尽己之心（己欲立而立人），"恕"是推己及人（己所不欲，勿施于人）。忠恕之道构成了"仁"的方法论。

**④ 中庸**——"过犹不及"，"执其两端而用其中"。中庸不是平庸折中，而是不偏不倚、恰到好处的智慧。

**⑤ 有教无类**——孔子打破"学在官府"的传统，开创私学，主张人人都有受教育的权利。"因材施教"是其教学方法。

**重要概念**：君子小人、义利之辨、学而时习之、温故知新

### 孟子（约公元前372-前289）——儒家"亚圣"

**① 性善论**——人性本善，犹如水之就下。"人皆有不忍人之心"，看到小孩要掉进井里，任何人都会产生恻隐之心，这不是出于功利计算，而是天性。性善论为儒家道德哲学提供了人性论根基。

**② 四端说**——恻隐之心（仁之端）、羞恶之心（义之端）、辞让之心（礼之端）、是非之心（智之端）。四端如同人的四肢，是与生俱来的道德萌芽，需要培养扩充。

**③ 仁政与王道**——"以不忍人之心，行不忍人之政，治天下可运之掌上"。孟子主张实行仁政，反对霸道（以力服人），推崇王道（以德服人）。经济上主张"制民之产"，让百姓有恒产才有恒心。

**④ 民贵君轻**——"民为贵，社稷次之，君为轻。"这一思想在中国古代政治哲学中极为激进，为后世民本思想奠定基础。

**⑤ 养浩然之气**——"我善养吾浩然之气"，这是一种至大至刚、配义与道的道德精神力量。

---

### 老子（约公元前571-约前471）——道家学派创始人

**① 道**——"道可道，非常道"。道是宇宙的本源和规律，无形无名，先天地而生。"道生一，一生二，二生三，三生万物。"道是一切存在的根源，但又不可言说。

**② 无为**——"道常无为而无不为"。无为不是什么都不做，而是不妄为、顺其自然。"无为而治"，统治者不要过多干预，让百姓自化、自正、自富、自朴。

**③ 自然**——"人法地，地法天，天法道，道法自然。"自然是道的根本法则，一切应当顺应万物的本然状态。

**④ 柔弱胜刚强**——"天下莫柔弱于水，而攻坚强者莫之能胜。""反者道之动，弱者道之用。"老子认为柔弱的表象下往往蕴含着最大的力量。

**⑤ 小国寡民**——理想社会是"小国寡民，使民有什伯之器而不用"，百姓"甘其食，美其服，安其居，乐其俗"，回归质朴自然的生活状态。

### 庄子（约公元前369-前286）——道家集大成者

**① 逍遥游**——追求绝对的精神自由。"北冥有鱼，其名为鲲……化而为鸟，其名为鹏。"庄子通过大鹏展翅、列子御风等寓言，层层递进，最终达到"无己、无功、无名"的逍遥境界，即精神与道合一的绝对自由。

**② 齐物论**——万物齐一。"天地与我并生，而万物与我为一。"庄子主张破除是非、美丑、大小、生死等的对立，认识到一切差别都是相对的。"方生方死，方死方生"，从道的角度看，万物没有本质区别。

**③ 无用之用**——"人皆知有用之用，而莫知无用之用也。"一棵大树因为"不材"（不成材）而免于被砍伐，得以终其天年。庄子以此说明，被世人认为"无用"的东西，恰恰可能有更大的用处——保全自身。

**④ 庖丁解牛**——通过庖丁宰牛的寓言，说明"依乎天理"、"因其固然"，顺应自然规律可以达到出神入化的境界。"以无厚入有间，恢恢乎其于游刃必有余地矣。"

**⑤ 庄周梦蝶**——"不知周之梦为蝴蝶与，蝴蝶之梦为周与？"探讨真实与虚幻的界限，物我两忘的境界。

### 墨子（约公元前468-前376）——墨家学派创始人

**① 兼爱**——"视人之国若视其国，视人之家若视其家，视人之身若视其身。"与儒家的"爱有差等"不同，墨子主张无差别地爱所有人。兼爱是墨子思想的核心。

**② 非攻**——反对侵略战争。墨子不是反对一切战争，而是反对"攻伐无罪之国"的不义之战。他本人曾"止楚攻宋"，通过辩论阻止楚国攻打宋国。

**③ 尚贤**——"官无常贵，而民无终贱。"主张打破世卿世禄制度，任人唯贤，不论出身。

**④ 天志与明鬼**——墨子相信天有意志、鬼神存在。天志是"兼相爱、交相利"的意志，鬼神能赏善罚暴。这为他的政治主张提供了超越性的权威依据。

**⑤ 节用与节葬**——反对奢侈浪费和厚葬久丧，主张节约资源用于民生。这一点与儒家形成鲜明对立。

### 荀子（约公元前313-前238）——儒家最后一位大师

**① 性恶论**——"人之性恶，其善者伪也。"与孟子截然相反，荀子认为人性本恶，人的自然欲望若不加以节制，必然导致争夺和混乱。善不是天生的，而是后天教化和人为努力的结果。

**② 化性起伪**——"伪"即人为。通过礼义法度的教育和规范，改变人的恶的本性，使之向善。"涂之人可以为禹"——普通人通过努力都可以成为圣人。

**③ 礼法并重**——荀子既强调礼（道德教化），也重视法（制度规范）。"隆礼尊贤而王，重法爱民而霸。"这一思想对后来法家产生重要影响，也奠定了汉以后"阳儒阴法"的政治传统。

**④ 明于天人之分**——"天行有常，不为尧存，不为桀亡。"自然界有其客观规律，与社会人事没有必然联系。这是中国古代思想中非常罕见的唯物主义倾向。

**⑤ 制天命而用之**——人不应只是顺从天命，而应"制天命而用之"——认识自然规律并利用它为人类服务。

### 韩非子（约公元前280-前233）——法家集大成者

**① 法、术、势**——法（成文的法律制度）、术（君主驾驭臣下的权术）、势（君主的权力和地位）三者结合，缺一不可。韩非认为，治国不能靠仁义道德，而要靠严明的法制和权术。

**② 人性自利**——韩非继承荀子的性恶论，进一步认为人与人之间完全是利害关系。"医善吮人之伤，含人之血，非骨肉之亲也，利所加也。"卖棺材的希望人死得多，不是因为他们心恶，而是利益驱动。

**③ 历史进化论**——"世界则事异，事异则备变。"韩非主张历史是发展的，不同的时代需要不同的治理方式。"今有构木钻燧于夏后氏之世者，必为鲧禹笑矣。"因此必须"因时变法"。

**④ 五蠹**——韩非认为社会上有五种"蛀虫"：学者（儒家）、言谈者（纵横家）、带剑者（游侠）、患御者（逃避兵役者）、商工之民。这些人不事生产、危害社会，应当取缔。

## 宋明理学（960-1644）

### 朱熹（1130-1200）——理学集大成者

**① 理气论**——"理"是宇宙的本体和规律，"气"是构成万物的物质材料。"理在气先"——理是先在的、永恒的；"理生气"——理通过气化生万物。"理一分殊"——同一个"理"体现在万事万物中，但表现各不相同。

**② 格物致知**——"即物而穷其理"。通过接触和研究具体事物，逐步认识其中的"理"，积累到一定程度就会豁然贯通，认识全体之理。这是一种理性主义的认识论。

**③ 天理人欲**——"存天理，灭人欲。"天理是人的道德本性，人欲是过度的私欲。不是要消灭一切欲望，而是要以天理节制人欲，使其符合道德规范。

**④ 大学章句**——朱熹将《大学》从《礼记》中抽出，重新编排，补写"格物致知"传文，使之成为理学的纲领性文献。他系统注释了《四书》，成为后世科举考试的官方标准。

---

### 王阳明（1472-1529）——心学集大成者

**① 心即理**——"心外无理，心外无物，心外无事。"与朱熹"理在物中"不同，王阳明认为理不在外物，而在内心。"物理不外于吾心"，宇宙的道理都在人的本心中。

**② 知行合一**——"知是行的主意，行是知的功夫；知是行之始，行是知之成。"知行不是两件事，而是一件事的两个方面。真知必然付诸行动，知而不行等于不知。这一思想极富实践精神。

**③ 致良知**——"良知"是每个人心中先天具有的道德判断力，不学而知、不虑而能。"致良知"就是把内心的良知扩充到万事万物上，使一切行为都符合良知的要求。这是王阳明心学的总纲领。

**④ 龙场悟道**——王阳明被贬贵州龙场，在极端困苦的环境下，一天夜里忽然大悟："圣人之道，吾性自足，向之求理于事物者误也。"——这就是著名的"龙场悟道"，标志着心学体系的确立。

**⑤ 四句教**——"无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物。"这四句话概括了王阳明心学的完整体系。

## 中国哲学主要流派总结

| 学派 | 创始人 | 核心主张 | 关键词 |
|:----:|:------:|:---------|:-------|
| 儒家 | 孔子 | 仁、礼、中庸 | 仁义礼智信 |
| 道家 | 老子 | 道法自然、无为 | 道、自然、无为 |
| 墨家 | 墨子 | 兼爱、非攻 | 兼爱、尚贤 |
| 法家 | 韩非子 | 法、术、势 | 以法治国 |
| 理学 | 朱熹 | 理气论、格物致知 | 存天理灭人欲 |
| 心学 | 王阳明 | 心即理、知行合一 | 致良知 |

**中国哲学的基本特征**：重伦理、重实践、重内在修养。与西方哲学注重知识体系和逻辑论证不同，中国哲学更关注"如何做人"、"如何治世"的问题。
